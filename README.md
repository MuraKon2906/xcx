# XCX

Skip the IDE. Spin up a full C++ project in seconds!

---

## Usage

```bash
xcx <command> [options]
```

---

## Commands

* `new <project_name>` – Create a new C++ project
* `run` – Run the project
* `build` – Build the project
* `add` – Add libraries
* `--help` – Show help

---

## Example

```bash
xcx new MyProject
```

---

## Requirements

* C++17+
* CMake 3.20+
